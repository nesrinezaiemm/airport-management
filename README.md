# Airport Management System

##  Project Description
AirportManagement is a .NET 8.0 application designed to manage airport activities in a structured and organized way.  
The project focuses on domain modeling, software architecture, and good development practices.
##  Project Objectives

- Develop a structured airport management application.
- Apply Object-Oriented Programming (OOP) principles.
- Implement data persistence using Entity Framework Core.
- Organize the project using clean and maintainable code practices.
- Use Git for version control and collaboration.
